# The Last of Guss - Frontend

## Технологии
- React 18
- TypeScript (strict)
- Vite
- React Router
- Ant Design
- React Query
- Axios
- SCSS Modules

## Запуск

```bash
npm install
npm run dev
```

Приложение запустится на http://localhost:5173

Backend должен быть на http://localhost:3000 (настроено в Vite proxy)
